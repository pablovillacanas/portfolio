# Seguimiento de proyectos

Fuente de la verdad para la pagina de freelance (modo "cuenta propia"). Cada
proyecto es un bloque separado por `---`. Actualiza este fichero cuando
empieces, avances o termines un proyecto: el contenido del portfolio
(servicios, casos de exito, stack) se redacta y revisa a partir de aqui.

Campos por proyecto:

- `titulo`: nombre corto del proyecto / cliente
- `cliente`: nombre del cliente o "Personal" / "Confidencial" si no se puede mencionar
- `periodo`: `MES AAAA - MES AAAA` o `MES AAAA - Presente`
- `estado`: `activo` | `pausado` | `finalizado`
- `rol`: tu rol en el proyecto
- `stack`: lista separada por comas
- `descripcion`: 2-4 frases sobre que se hizo
- `impacto`: resultados / impacto para el cliente (metricas si las hay)
- `mostrar_en_portfolio`: `si` | `no` (por confidencialidad)
- `notas`: notas internas, aprendizajes, contexto que no va en la web
